  • DOM
  • fotbal.cz
  • .htacess router

Football clubs have always wanted to show results and tables on their websites. Nobody likes filling out match results when the results are available at official football website. Unfortunately FAČR does not provide API, so Scortes parses HTML pages, transforms results and offers customized templates accessible via URL.

BN-PHP

Homepage: bitbucket.org/zdenekdrahos/bn-php
  • PHP
  • OOP
  • BC Math
  • Shunting Yard
  • Infix/Postfix notation
  • New BSD

Big Number data type which can be used as replacement of inaccurate float. Now you don't have to worry about floating point precision problem. It use BC Math library for basic immutable Number but it also contains expression evaluator with implemented Shunting Yard algorithm and Postfix algorithm.
