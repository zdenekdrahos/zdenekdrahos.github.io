What is Treewec?

Treewec is simple but powerful tool for creating and managing websites without database. This opensource PHP app can be used as CMS, editor for static websites, wiki software or web-based file manager. It's easy to use and it's completely customizable to all your demands. Requires PHP >= 5.3

How it works?

  1. You choose (non-public) directory and set up basic web settings
  2. Treewec arranges displaying content of the directory on the web!

Why use Treewec?

  • Easy to install and use.
  • No database nor XML files.
  • One installation for many websites.
  • Supports multi-language, pretty URL addresses, UTF-8.
  • Manage error pages (404, ...).
  • Different ordering of files.
  • Completely customizable via themes and templates.
