Interface ITokenizer

Used for breaking a text (in this case mathematical expression) into operators, operands and brackets. Parsed tokens are available only via iterator. Interface is also child of interface @see \IteratorAggregate. Used in @see ShuntingYardParser for tokenizing mathematical expressions.
