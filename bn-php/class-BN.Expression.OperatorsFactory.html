public BN\Expression\Operators
# getOperators( array $symbols )

Gets selected operators. Available symbols are:

  • '^' for power
  • '*' for multiply
  • '/' for divide
  • '\' for quotinent (integer division without remainder)
  • '+' for addition
  • '-' for subtract
  • '%' for modulo,
  • 'sqrt' for square root
  • 'round' for round
  • 'ceil' for round up
  • 'floor' for round down
  • 'roundTo' for round to specified number
  • 'abs' for 'abs'
